4C> THIS FUNCTION LOOKS FOR ALL STACKED DATA EVENTS FOR A

7C> EVENTS ARE ACCUMULATED AND RETURNED TO THE CALLING PROGRAM WITHIN
8C> ARRAY USR. THE VALUE OF THE FUNCTION ITSELF IS THE TOTAL NUMBER
