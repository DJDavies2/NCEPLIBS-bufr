4C> THIS SUBROUTINE IS CALLED BY BUFR ARCHIVE LIBRARY SUBROUTINE
5C> UFBRW WHENEVER IT CAN'T FIND A MNEMONIC IT WANTS TO WRITE WITHIN THE
6C> CURRENT SUBSET BUFFER. IT LOOKS FOR THE MNEMONIC WITHIN ANY
7C> UNEXPANDED "DRS" (STACK) OR "DRB" (1-BIT DELAYED REPLICATION)
8C> SEQUENCES INSIDE OF THE PORTION OF THE SUBSET BUFFER BOUNDED BY THE
9C> INDICES INV1 AND INV2. IF FOUND, IT EXPANDS THE APPLICABLE "DRS" OR
10C> "DRB" SEQUENCE TO THE POINT WHERE THE MNEMONIC IN QUESTION NOW
11C> APPEARS IN THE SUBSET BUFFER, AND IN DOING SO IT WILL ALSO RETURN
12C> A NEW VALUE FOR INV2.
13C>

43C> 0 = NOT FOUND
44C> INV2 - INTEGER: IF INVN = 0, THEN INV2 IS UNCHANGED FROM ITS
45C> INPUT VALUE. OTHERWISE, IT CONTAINS THE REDEFINED
46C> ENDING INDEX OF THE PORTION OF THE SUBSET BUFFER
47C> CURRENTLY BEING PROCESSED BY UFBRW, SINCE EXPANDING A
48C> DELAYED REPLICATION SEQUENCE WILL HAVE NECESSARILY
49C> INCREASED THE SIZE OF THIS BUFFER.

51C> REMARKS:
