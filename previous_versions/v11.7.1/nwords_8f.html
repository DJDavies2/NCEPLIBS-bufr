THIS FUNCTION ADDS UP THE COMPLETE LENGTH OF THE DELAYED REPLICATION SEQUENCE BEGINNING AT INDEX N OF THE DATA SUBSET.

PROGRAM HISTORY LOG: 1996-10-09 J. WOOLLEN – ORIGINAL AUTHOR 1999-11-18 J. WOOLLEN – ADDED DOCUMENTATION 2014-12-10 J. ATOR – USE MODULES INSTEAD OF COMMON BLOCKS

USAGE: NWORDS (N, LUN) INPUT ARGUMENT LIST: N - INTEGER: INDEX TO START OF DELAYED REPLICATION SEQUENCE LUN - INTEGER: I/O STREAM INDEX INTO INTERNAL MEMORY ARRAYS

OUTPUT ARGUMENT LIST: NWORDS - INTEGER: COMPLETE LENGTH OF DELAYED REPLICATION SEQUENCE WITHIN DATA SUBSET

REMARKS: THIS ROUTINE CALLS: None THIS ROUTINE IS CALLED BY: INVMRG Normally not called by any application programs.
